# Etude-jeu-de-donnees-sur-les-maladies-cardiaques
Les maladies cardiaques sont une des principales causes de mortalité sur 9 500 individus. 

Objectif : identifier les facteurs déterminants
Quels facteurs augmentent significativement le risque de maladie cardiaque ? 

Mesures créees : 
Nombre d’individus, Taux de maladie cardiaque (Heart Disease Yes %)

Colonnes créees : 
Catégorie de Sommeil,  Catégorie Cholestérol, Catégorie Niveau Cholestérol, Catégorie CRP, Tranche d’âge,  Catégorie Homocysteine, Catégorie Hypertension, Diabète (selon glycémie à jeun), IMC
